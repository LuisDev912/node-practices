# node-practices

This is a repository that contains small **Node.js exercises** focused on learning backend fundamentals without any frameworks.

## Goals

- Understand Node.js as a runtime.
- Learn how the file system works.
- Build HTTP servers from scratch.
- Handle JSON APIs and environment variables.

## Tech Stack

- Node.js
- JavaScript (ES Modules)

## Project Structure

Each folder contains a focused practice with its own README and executable code.

## How to Run

1. Clone the repository
2. Navigate to any practice folder
3. Run `node index.js`

This repository is part of my backend learning process and is meant for educational purposes.
